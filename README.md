# DYNAPEX - React 웹사이트

데스크톱·모바일 반응형 지원 React 기반 DYNAPEX 공식 홈페이지입니다.

## 기술 스택

- **React 19** + **Vite 8**
- **React Router** (라우팅)
- **CSS Modules** (헤더 스타일)
- **Context API** (언어 전환 EN/KR)

## 제외된 콘텐츠

- Aneurysm (뇌동맥류)
- Liver (개발 중)

## 포함된 AI 제품

| 카테고리 | 제품 |
|---------|------|
| Oncology | Glioblastoma, Brain Metastasis, Head and Neck |
| Vascular | Stroke |
| Degenerative | ARIA, Parkinson's disease |
| Demyelinating | Multiple Sclerosis |

## 개발

```bash
cd web
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

빌드 결과물은 `dist/` 폴더에 생성됩니다.
